# 📊 Customer Behavior Analysis Dashboard

## 📌 Project Overview

This project focuses on analyzing customer shopping behavior to uncover meaningful business insights and support data-driven decision-making.

Using **Python, SQL, and Power BI**, the project explores customer demographics, purchasing patterns, subscription behavior, and product performance to identify trends that can improve customer engagement, sales, and marketing strategies.

---

## 🎯 Objectives

- Analyze customer purchasing behavior across different demographics.
- Identify top-performing product categories and customer segments.
- Understand the impact of subscription status on customer behavior.
- Generate actionable insights to optimize business strategies.

---

## 🛠️ Tools & Technologies

- **Python (Pandas, NumPy)** – Data cleaning and preprocessing
- **SQL (PostgreSQL/MySQL)** – Data analysis and business queries
- **Power BI** – Interactive dashboard creation
- **Excel** – Initial data exploration

---

## 🔄 Project Workflow

1. Data Cleaning and Transformation using Python.
2. Exploratory Data Analysis and Business Insights using SQL.
3. Solved 10 business-oriented SQL questions.
4. Built an interactive Power BI dashboard.
5. Created a detailed report and presentation with recommendations.

---

## 📈 Dashboard Highlights

- Total Customers Analysis
- Average Purchase Amount
- Customer Subscription Analysis
- Revenue by Product Category
- Sales by Category
- Revenue and Sales by Age Group
- Interactive Filters

---

## 💡 Key Insights

- Clothing category generated the highest revenue and sales.
- Young adults contributed the highest revenue among all age groups.
- Most customers were non-subscribers, indicating opportunities for loyalty programs.
- Customer demographics significantly influence purchasing behavior.
